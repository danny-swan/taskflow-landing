# TaskFlow Landing

Публичная посадочная страница приложения [TaskFlow](https://github.com/danny-swan/taskflow-app) — удобного менеджера задач для Windows.

- Прод: [yourtaskflow.app](https://yourtaskflow.app)
- Хостинг: GitHub Pages
- Стек: чистый HTML/CSS/JS, без сборки

## Обновление

Отредактировать `index.html`, закоммитить в `main` — GitHub Pages задеплоит автоматически через 30-60 секунд.

## Скачать TaskFlow

Кнопка «Скачать» на лендинге тянет актуальный `x64-setup.exe` через GitHub Releases API. Прямая ссылка на последнюю версию:

```
https://github.com/danny-swan/taskflow-app/releases/latest/download/TaskFlow_<VERSION>_x64-setup.exe
```
