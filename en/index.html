A comfortable working tool. Adapts to you.

TaskFlow doesn't try to replace Notion or Jira. It focuses on the essentials — a simple, clear interface that's comfortable to use every day.

Kanban and list

Switch between list and Kanban board. Drag & drop between columns. Five default statuses — «Important», «Today», «In progress», «To do», «Completed» — plus the ability to configure your own to match your process.

Calendar and deadlines

Overdue and today's tasks are highlighted automatically. See workload by month, assign deadlines, group by category — everything is visible at a glance, nothing gets lost.

Dashboard and statistics

How many tasks you closed this week, where you're stuck, which category eats the most time. Data to make decisions with — not for decoration.

Russian and English

Full interface localization. Switch language directly in the app — no restart needed. Equally comfortable in Russian and English, no translation losses.

Categories and priorities

Custom color categories for projects, priorities, and contexts. One-click quick filters — show only what matters right now.

JSON export and import

Your data is yours. Export everything to JSON with one click, import on another device. No lock-in.

⚠ How to install TaskFlow on macOS

The build is not signed with an Apple certificate, so macOS Gatekeeper will show a warning on first launch. That's normal — here's how to open the app.

Option 1 — via Finder (easier)
  1. Open the downloaded .dmg and drag TaskFlow into the Applications folder
  2. Open the Applications folder, right-click (or Control-click) TaskFlow → «Open»
  3. In the confirmation dialog click «Open» again
  4. Done — from now on the app opens with a normal double-click
Option 2 — via Terminal
  1. Install the app into Applications
  2. Open Terminal and run: xattr -d com.apple.quarantine /Applications/TaskFlow.app
  3. After that the app opens without any warnings
ℹ️ The app is safe — the source is open and builds run via GitHub Actions. There is no signature yet because Apple Developer ID costs $99/year; we're gathering macOS users before buying.
